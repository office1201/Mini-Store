# 🌼프로젝트명: 미니 의류 스토어

> 목적: 웹 프론트엔드 기초 다지기

## 🔨 기술 스택

- HTML5
- CSS3
- Banila JS(ECMA Script 6+)

## 🔨 구현 기능

- `HTML`
  - Semantic tac를 사용한 마크업
  - Entity code를 활용한 특수 문자(<>) 마크업
- `CSS`
  - 전역 변수를 활용한 CSS 스타일링
  - calc() 연산 함수를 통한 CSS 스타일링
  - 버튼 에니메이션 추가
- `Banila JS`
  - JSON 객체를 활용한 데이터 통신 및 자바스크립트 기능 구현
  - filter() 내장 함수를 통한 필터링 기능 구현
